# Katriel & Freitas Advogados — site institucional

Site estático, responsivo e pronto para publicação no GitHub e na Vercel.

## Posicionamento desta versão

A comunicação foi ampliada para apresentar o escritório como uma banca de atendimento a pessoas físicas e famílias, profissionais autônomos, empresas e organizações.

O site destaca Direito Previdenciário e Assistencial, especialmente BPC/LOAS, sem retirar as áreas Civil, do Consumidor, Empresarial, Administrativo e Tributário.

## Publicação na Vercel

1. Extraia o ZIP.
2. Crie um repositório no GitHub e envie todos os arquivos desta pasta para a raiz.
3. Na Vercel, selecione **Add New → Project** e importe o repositório.
4. Mantenha **Framework Preset: Other**, sem comando de build.
5. Publique o projeto.

## Arquivos principais

- `index.html`: conteúdo e estrutura da página.
- `styles.css`: identidade visual e responsividade.
- `script.js`: menu, animações e formulário para WhatsApp.
- `assets/`: logotipo, favicon e imagem de compartilhamento.
- `vercel.json`: cabeçalhos e regras de publicação.

## Antes de publicar

Revise os serviços previdenciários e assistenciais descritos na página e confirme que correspondem às áreas efetivamente atendidas pelo escritório. Confirme também os números de WhatsApp e as informações profissionais dos sócios.

O formulário não armazena dados: ele apenas monta a mensagem e abre uma conversa no WhatsApp.
